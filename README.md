# Rust Learning Guide for Beginners

Welcome to the comprehensive Rust learning guide! This guide covers all fundamental concepts you need to understand Rust programming.

## 📚 Learning Path

### 1. **01_basics** - Hello Rust!
Introduction to Rust, setting up, and your first program.

### 2. **02_variables_data_types** - Variables & Types
Learn about variables, mutability, and Rust's type system.

### 3. **03_functions** - Functions
Function declarations, parameters, return values, and expressions.

### 4. **04_ownership_borrowing** - Ownership & Borrowing ⭐ (Core Concept!)
The most important Rust concept - memory management without garbage collection.

### 5. **05_structs_enums** - Structs & Enums
Custom data types and creating complex structures.

### 6. **06_pattern_matching** - Pattern Matching
The powerful `match` statement and pattern matching techniques.

### 7. **07_collections** - Collections
Vectors, HashMaps, and other collection types.

### 8. **08_error_handling** - Error Handling
Result<T, E>, Option<T>, and error management.

### 9. **09_iterators** - Iterators & Closures
Functional programming patterns in Rust.

### 10. **10_traits_generics** - Traits & Generics
Create reusable code with generics and define behavior with traits.

## 🎯 How to Use This Guide

Each folder contains:
- **THEORY.md** - Explanation of concepts with diagrams
- **examples/** - Runnable Rust code examples
- **exercises/** - Practice problems for you to solve

## ✅ Prerequisites

- Install Rust: https://www.rust-lang.org/tools/install
- Command: `rustup` and `cargo`

## 🚀 Running Examples

```bash
# Navigate to any example folder
cd RUST_LEARNING_GUIDE/01_basics/examples

# Run an example
cargo run
```
